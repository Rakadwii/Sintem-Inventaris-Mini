## Cara Menjalankan:
1. Pindahkan folder proyek ke `htdocs`.
2. Import file `inventaris_db.sql` ke phpMyAdmin.
3. Sesuaikan konfigurasi di `koneksi.php`.
4. Akses melalui browser 
5. Tampilan Pertama pada aplikasi adalah Halaman Login bisa langsung menginputkan (usernama:admin) dan (password:admin123)
6. Setelah berhasil login akan ada 2 Menu yaitu Kelola Barang dan Kelola Kategori di bagian atas ada Button "Kelola Barang" dan "Kelola Kategori" untuk berpindah halaman
7. Jika di menu Kelola Barang disitu pengguna bisa melakukan CRUD untuk Barangnya
8. Jika di menu Kelola Kategori pengguna juga dapat melakukan CRUD Kategori
9. Lalu jika ingin Logout bisa klik button dipojok kanan atas
10. Jika ingin mencari Barang  bisa menulikan namanya di bagian Cari nama barang.. yang ada di atas dan klik button "Cari" kalau mau mereset bisa klik Button "Reset"
